### Move submodule
Having not fun trying to move the submodule.
First couple of goes made a complete twisted mess of the local checkout.
(...just managed to revert to working order...)
Git docs look they have options for changing submodule branch and url, but not path afaics.
So try three, will deinit the module entirely and readd it at the new path.

	git submodule deinit "yearclock/[html-common]"


Readd:

	git submodule add "../html-common" [html-common]
	git config --local submodule."[html-common]".url "../html-common/"

Fiddling around a bit, but looks like its back running again.

There are definitely some gotchas to do with submodule handling.
In one case I had to preemptively deinit and re-init before I could even merge a branch.
